# Roadmap

This roadmap is intentionally small and practical.

## v0.1.0 - First usable OpenCode bridge

- pip-installable MCP server
- OpenCode stdio config examples
- workspace status tool
- deterministic context pack
- replay digest
- safety contract check
- local GitHub summary pack
- Hugging Face export pack draft

## v0.2.0 - Stronger Rust workspace support

- safer Rust audit implementation
- better Cargo workspace detection
- structured Cargo metadata parsing
- fixture Rust project for tests
- clearer Windows error messages

## v0.3.0 - Release-ready collaborator bundle

- GitHub release workflow
- wheel artifact upload
- checksums
- offline install bundle
- optional bundled `ctxt` instructions

## v0.4.0 - Community integrations

- OpenCode setup screenshots
- Hugging Face Space demo
- example videos or GIFs
- contributor issue labels
- public examples from small Rust repos

## Principles

Every roadmap item must preserve:

- no local LLM requirement
- no model downloads
- read-only default
- deterministic output where practical
- clear audit trail
